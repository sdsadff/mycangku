程序题

JavaBean（10分）

JavaBean 必须满足以下三个条件：

  • 私有字段（private fields）
  • 公共的无参构造方法（public no-argument constructor）
  • 为每个字段提供公共的 getter 和 setter 方法（public getter and setter methods）

示例代码：

package com.example.model;

public class Person {
    private String name;
    private int age;

    // 无参构造方法
    public Person() {
    }

    // 有参构造方法
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Getter 和 Setter 方法
    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        return age;
    }

    public void setAge(int age) {
        this.age = age;
    }
}
